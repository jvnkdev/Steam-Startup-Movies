# Security Policy

This repository contains custom Steam startup movie files (`.webm` videos and `.jpg` thumbnails) and documentation. It does not contain executable code, scripts, or software with runtime dependencies.

## Reporting a Vulnerability

If you believe you've found a security issue related to this repository (for example, a malicious or tampered file), please open an issue on this repo or contact the repository owner directly through GitHub.

## Scope

- Video and image files in this repo are provided as-is for personal customization use with Steam.
- Always download files only from the official source (this repository) to avoid tampered or malicious copies circulating elsewhere.
- This project does not collect, transmit, or process any user data.

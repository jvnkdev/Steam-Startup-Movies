                Tutorial

Hello there, these are my picks of Steam movies!To put them in Steam, follow my lead.

1. Find Steam located in C:Program Files (x86)Steam. If not, check D:.

2. Find the config folder and enter it.
 
3. Download my picks and extract them in config.
  
4. Enter Steam Big Picture mode by pressing the monitor button up there.

5. Go into settings in Big Picture mode, press customization, and then select the one you like

6. Enjoy.

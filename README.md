# Steam Startup Movies

A small collection of custom Steam boot/startup movies — swap out the default Steam intro for something more fun.

## Included Movies

- `cyberpunk_2077_system_reboot.webm`
- `m4XR2QVn3IJo4kPPu7YvFaitBW0WtQCoePTjEmhU.webm`
- `ssstik.io_1783354193668.webm`
- `retro_crt_glitch_boot.webm` — original retro CRT/glitch intro (fully custom, no copyrighted material)

Each video comes with a matching `.jpg` thumbnail preview.

## Installation

1. **Locate your Steam folder.** By default, this is:
   ```
   C:\Program Files (x86)\Steam
   ```
   If it's not there, check other drives (e.g. `D:\Steam`).

2. **Open the `config` folder** inside your Steam directory.

3. **Download and extract** the contents of this repo's `uioverrides/movies` folder into `config`.

4. **Launch Big Picture Mode** (the monitor icon in the top-right of the Steam client).

5. **Open Settings → Customization**, then select the startup movie you'd like to use.

6. Enjoy your new boot animation! 🎬

## Notes

- Files are provided in `.webm` format, which Steam expects for custom startup movies.
- Don't rename the video files — Steam's customization menu references them by their exact filenames.
